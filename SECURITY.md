# Security Policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a vulnerability

Please report suspected vulnerabilities privately via
[GitHub security advisories](https://github.com/dominicsayers/name_tamer/security/advisories/new).
Please do not open a public issue for security reports.

You should normally receive a response within a week. If the report is
confirmed, a fix will be released as soon as practical and credited to you
unless you prefer otherwise.
